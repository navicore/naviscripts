personal scripts for vim and tmux and anything else I use
------------

* install or update after doing a git pull

`git pull`

`./install.sh`

* update the repo from the installed files

`./update_git.sh`

`git commit -a -m np`

`git push`

* create tar for non-git install (remote)

`tar_dot_files.sh`

look for the tar in ~
